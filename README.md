# eCommerance-Case
React-Native E-Commerance Mobil App

## Instalation
In the project root path

`````
>npm install
>npm run start (for Metro Server)

`````

Run emulator or connect your phone to your computer

### For Android

````
npm run Android
````

### For Android

````
npm run Android
````

The application will run automatically
